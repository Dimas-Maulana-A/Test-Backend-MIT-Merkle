# Buku Tamu

## Project setup
```
npm install
```

### Run server
```
nodemon server
```

### Url Buku Tamu
Mengambil semua data
```
GET /tamu
```
Menambah data
```
POST /tamu
```
ADMIN
GET /admin/tamu
```
Menghapus data
```
DELETE /admin/tamu:id
```

### Url path admin
Menambahkan Admin
```
POST /admin
```
Login
```
POST /login
```
Mengambil semua data
```
GET /admin
```
Mengambil data berdasarkan id
```
GET /admin/:id
```
